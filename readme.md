# Bitcoin Website

A BTC website built on React, HTML5, CSS3, and Node.

## Getting Started

Download or clone this repo.

### Prerequisites

Install the required packages:

```
npm install
```

### Build

First build the application:

```
npm run build
```

Then start the webpack server:

```
npm run start
```

And then navigate to the site: 

```
http://localhost:8080
```
